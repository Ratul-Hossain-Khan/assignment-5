# Github Issues Tracker

## 1 What is difference between var let const

var → function scoped and can be redeclared.

let → block scoped and can be updated but not redeclared.

const → block scoped and cannot be updated or redeclared.

---

## 2 Spread Operator (...)

Spread operator allows expanding elements from arrays or objects.

Example:

const arr=[1,2,3]
const newArr=[...arr,4]

---

## 3 map filter forEach

map → returns new array

filter → returns filtered array

forEach → only loops but returns nothing

---

## 4 Arrow Function

Short syntax for writing functions.

Example:

const sum=(a,b)=>a+b

---

## 5 Template Literals

Allows embedding variables using backticks.

Example:

const name="John"

console.log(`Hello ${name}`)
